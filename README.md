# HR Database Management System

## Overview

A database management system designed to manage employee information, departments, attendance records, and payroll operations.

This project was developed as part of academic coursework in the Computer Science program.

---

## Features

- Employee Management
- Department Management
- Attendance Tracking
- Payroll Management
- Data Retrieval and Reporting

---

## Technologies Used

- SQL
- Relational Database Design

---

## Concepts Applied

- Database Design
- Entity Relationship Modeling (ERD)
- Normalization
- SQL Queries
- Joins
- Triggers

---

## Project Objectives

The main objective of this project is to demonstrate database design principles and efficient data management techniques for human resources operations.
